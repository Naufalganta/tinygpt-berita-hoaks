# TinyGPT Berita Hoaks

Implementasi TinyGPT sederhana menggunakan Transformer dan SentencePiece untuk menghasilkan teks berita hoaks berbahasa Indonesia.

## Dataset
Corpus disusun dari kumpulan artikel pemeriksaan fakta.

## Teknologi
- Python
- PyTorch
- SentencePiece
- Transformer

## Cara Menjalankan

Install:
pip install torch sentencepiece

Run:
python tinygpt.py

## Hasil
Model dilatih menggunakan corpus ±2347 kata dan menghasilkan teks sederhana berdasarkan pola corpus.
